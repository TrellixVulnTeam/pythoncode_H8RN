













HOLA Outlook Plugin 1.0
用户使用手册



















HOLA Outlook Plugin 是一款安装在用户电脑上的客户端程序，它可以在MS Outlook
与HOLA 系统之间建立连接，并帮助用户完成如下操作：

	邮件归档（邮件内容及附件归档）— 用户可以将MS Outlook的邮件（包含附
件）归档到HOLA系统。只需选中相应的邮件（可包含附件）即可轻松将其归
档到HOLA系统不同的文件夹。

	联系人备份 — 可将联系人从MS Outlook备份到HOLA系统（单向备份）。
联系人备份可以以防万一您无法从您的电脑查阅联系人信息，也可以直接登陆
HOLA系统，从HOLA系统查阅所有联系人及其相关信息。

1.	HOLA Outlook Plugin的安装
HOLA Outlook Plugin可以安装在32位及64位的如下操作系统下：

	Windows 7
	Windows Vista
	Windows XP

显然，为了正常使用HOLA Outlook Plugin，需要您的电脑上已安装了MS Office Outlook。
HOLA Outlook Plugin支持如下Outlook 版本：

	MS Office Outlook 2010
	MS Office Outlook 2007

另外，为了保证HOLA Outlook Plugin能连接上HOLA 系统，需要满足如下要求：

	服务器上安装的是HOLA 6.1.1或者更高的版本（标准版或者企业版）
	HOLA 系统的Web Service已成功发布
	HOLA 系统的HOLA Outlook Plugin功能已启用（通过License授权启用，与
HOLA Sync使用相同的license）
	用户已被授权允许使用HOLA Outlook Plugin连接HOLA 系统（可在管理员端 
—Sync用户管理进行设置）

如果HOLA系统尚未安装或者您不确认HOLA Web Service是否已经发布，请阅读
《HOLA系统安装手册》或者联系您的系统管理员。

如果您不清楚如何授予/移除用户使用HOLA Outlook Plugin连接HOLA系统的权限，
请阅读《HOLA系统管理手册》之站点管理篇。

如果您没有启用HOLA Outlook Plugin功能的License，请联系我们，邮件地址：
info@deyatech.com。

3. 邮件归档（邮件内容及附件归档）
一旦HOLA Outlook Plugin安装成功并且初始化设置正确，您就可以很轻松的将Outlook
的邮件归档到HOLA了。

在归档过程中，邮件正文会被转化成PDF格式并保存到指定的文件夹下。同时，HOLA
系统会自动为该PDF归档文件建立索引，以便日后可以对归档后的邮件的正文内容进
行搜索。

如果邮件中包含多个附件的话，您可以有选择的将附件归档到HOLA系统，附件将以原
格式进行保存。

1.	在Outlook里选择一个或者多个邮件，右击鼠标，在右键菜单里选择“上载到HOLA”。

到HOLA”。

4. 联系人备份
一旦HOLA Outlook Plugin成功安装并且初始化设置正确，您就可以着手将联系人从
Outlook备份到HOLA系统。

Outlook 的联系人备份到HOLA后，它们会被储存在HOLA“个人工作区”的“联系人”
单元里。


 

Outlook联系人备份后将存储在HOLA“个人工作区”的“联系人”单元。

HOLA Outlook Plugin只支持单向备份（从Outlook 备份到HOLA）。这意味着所有来
自Outlook的联系人将会被备份并存储在HOLA系统里，可是在HOLA系统里创建的
联系人并不会复制到Outlook里。HOLA Outlook Plugin作为一个Outlook联系人备份
的工具，为您在无法查阅电脑上Outlook联系人信息的情况下提供了另外个一个途径—
登陆HOLA系统查看。

由于Outlook和HOLA联系人的属性并不完全相同，所以个别情况下，Outlook联系人
某些信息不会被保存到HOLA。例如：HOLA联系人不支持联系人形象（图片），所以
假如在Outlook上是有联系人的图片的话，该图片将不会被保存到HOLA里，可是其余
的信息还是会被正确保存。一般情况下，联系人的信息是会从Outlook备份到HOLA的。

联系人备份

1.	找到Outlook的联系人单元，点击工具栏上的“备份到HOLA”，或者也可以点击
菜单栏的“动作”—“HOLA”—“备份到HOLA”。


 

在Outlook “联系人”单元，点击“备份到HOLA”按钮。

2.	弹出联系人备份进度条，从进度条的下方我们可以看到一共有多少联系人正在备份，
当前已经完成备份的联系人都有多少个。点击“取消”按钮，可以取消备份进程。
在备份的过程中，“确定”按钮不可用。


 

联系人备份进度条

3.	当进度条完成后，您可以点击“确定”按钮关闭进度条显示。联系人已成功备份，
如需检查备份情况，可以登陆HOLA系统，进入“个人工作区”的“联系人”单元
进行检查。


 

当备份进度结束后，点击“确定”按钮关闭进度条。

编辑或者删除已成功备份的联系人。

首次备份成功后，HOLA Outlook Plugin会在Outlook联系人和成功备份到HOLA上的
联系人之间建立关联，所以当我们编辑或者更新Outlook上某些联系人，点击 “备份
到HOLA”，HOLA上的联系人也会作相应的改变。

删除联系人也是一样的道理，如果您删除了Outlook上的个别联系人，之后点击“备份
到HOLA”，HOLA上相应的联系人也会被删除。

冲突处理

在首次备份过程中（或者是备份Outlook新创建的联系人的时候），可能会出现HOLA
系统已经存在相同的联系人（相同的姓名和邮件地址），这种情况下会出现提示弹窗，
要求您进行确认，以便进行下一步的操作，其中有两个选择：

•	覆盖 — 选择该选项，Outlook上的联系人信息会覆盖HOLA上相同的联系人信
息。
•	跳过 — 选择该选项，Outlook Plugin不做任何处理（Outlook上相同的联系人
不会备份到HOLA，HOLA上原有的联系人信息不改变）


 

联系人在HOLA中已存在提示弹出窗口包括覆盖和跳过选项。

每当HOLA联系人和Outlook的联系人存在冲突时，都会出现提示弹窗，其中提示弹窗
中包含“以下冲突是否执行相同的操作”的选项：

•	当勾上该选项，并且选择“覆盖”，当前以及后面出现的冲突都会执行“覆盖”
的操作。
•	当勾上该选项，并且选择“跳过”，当前以及后面出现的冲突都会执行“跳过”
的操作。
•	如果不勾选该选项，每当备份过程中出现冲突，都会出现提示弹窗。

5. HOLA Outlook Plugin的卸载
如果您决定要卸载HOLA Outlook Plugin，卸载向导会指导您很容易的完成卸载过程。

请注意，一旦HOLA Outlook Plugin卸载后，您将无法直接将邮件归档到HOLA，也无
法实现联系人的备份。

为了保证HOLA Outlook Plugin卸载的顺利进行，请在卸载前关闭Outlook。

1.	打开开始菜单，找到“控制面板”—“添加删除程序”。


 







 
打开开始菜单-“控制模板”
在“控制面板”，点击“添加/删除程序”。

2.	“添加删除程序”窗口打开后，在“当前安装程序”列表里找到HOLA Outlook Plugin，
选择“删除” 
 


 

选中HOLA Outlook Plugin，选择“卸载”。

3.	启动卸载向导，开始卸载前的准备。
 

 

卸载准备向导。

4.	卸载准备完成后，卸载向导会询问你是否确认卸载HOLA Outlook Plugin。


 

点击“是”开始Outlook Plugin的卸载。

5.	选择“是”开始卸载进程。
 


 

点击“是”开始Outlook Plugin的卸载。

6.	卸载完成后，会出现卸载成功的提示，点击“完成”关闭向导窗口。


 

.卸载完成后，会出现卸载成功的提示。

7.	卸载完成后， Outlook上HOLA Outlook Plugin的相关按钮和菜单选项就不再显示。


 

Outlook里HOLA Outlook Plugin 的相关按钮不再显示。.


 

Outlook里HOLA Outlook Plugin 的相关按钮（“动作”菜单或者右键菜单）不再显示。.
