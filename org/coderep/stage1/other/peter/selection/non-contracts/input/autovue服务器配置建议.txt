AutoVue 服务器配置推荐


一、	硬件要求

　　CPU： 四核英特尔® 至强® E3系列处理器 主频2.0GHz以上  1颗
　　内存：8G内存，1333MHz
　　硬盘：两块 300 SAS RAID 1
　　电源：两个热插拔高效570瓦PSU或两个高输出热插拔870瓦PSU
　　特性：热插拔硬盘；热插拔冗余电源；热插拔冗余冷却；群集支持
　　
二、	软件要求

　　Windows Server 2008 32bit/64bit
　　Windows Server 2008 R2 32bit/64bit
　　Windows Server 2012 32bit/64bit
　　
三、	客户端要求

　　Windows OSes:
　　Windows XP 64-bit
　　Windows Vista 64-bit 
　　Windows 7 64-bit
　　Windows 8 64-bit
　　Windows 10 64-bit
　　■Internet Explorer 7
　　■Internet Explorer 8
　　■Internet Explorer 9
　　■Internet Explorer 10
　　■Firefox ESR 24
　　■Chrome 21
　　MAC OS X 10.8
　　■Safari 5.01
　　■Firefox ESR 24
